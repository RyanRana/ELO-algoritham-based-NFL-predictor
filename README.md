# ELO-algoritham-based-NFL-predictor

An ELO Ranking system uses the ELO ranking system to rank NFL teams using historical data and then predecting output of games.
<br><br>
Install the following models: 
<br><br>
nflgame: 1.2.20
termcolor: 1.1.0
requests: 2.11.1
lxml: 3.5.0
